### reCSSet

----

Just an ordinary CSS reset.

### How to Use

```markup
<link rel="stylesheet" type="text/css" href="path/to/recsset.min.css">
```

or

```css
@import 'path/to/recsset.min.css';
```

### Bower

```bash
bower install recsset
```
